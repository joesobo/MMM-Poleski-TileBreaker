Basic mod for the Mushroom Madness game on Steam

Built using the Godot Mod Loader: https://wiki.godotmodding.com/

This mod adds a config for changing the time to break for each tile type. Config is located at `INSTALL_DIR/mod_configs/Poleski-TileBreaker/default.json`

```json
{
	"floor_break_time": 20.0,
	"glass_break_time": 20.0,
	"ice_break_time": 20.0
}
```
